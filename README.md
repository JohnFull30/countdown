# 🎉 Gender Reveal Countdown

This is a responsive, cinematic countdown app for gender reveal celebrations. Includes full-screen video background, confetti effects, and mobile-first design.

---

## 🚀 How to Run Locally

```bash
npm install
npm start


# Gender Reveal Countdown 🎉

## Dev Setup
- `npm install`
- `npm start` to run locally
- `npm run build:web` to build for GitHub Pages

## Deploy
- Use `./choose-script.sh` > Option 1
```
